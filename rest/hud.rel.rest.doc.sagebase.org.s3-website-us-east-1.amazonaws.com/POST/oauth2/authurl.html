Synapse REST API

POST /oauth2/authurl

The first step in OAuth authentication involves sending the user to authenticate on an OAuthProvider's web page. Use this method to get a properly formed URL to redirect the browser to an OAuthProvider's authentication page. Upon successful authentication at the OAuthProvider's page, the provider will redirect the browser to the redirectURL. The provider will add a query parameter to the redirect URL named "code". The code parameter's value is an authorization code that must be provided to Synapse to validate a user.
