V2WikiHistorySnapshot

org.sagebionetworks.repo.model.v2.wiki.V2WikiHistorySnapshot

A WikiHistorySnapshot contains basic information about an update to a WikiPage.

Field Type Description
modifiedOn STRING The timestamp when this modification/snapshot took place.
modifiedBy STRING The id of the user that made this modification.
version STRING The version of this snapshot in the wiki's history.
